How to Install WhatsApp Web Helper (Chrome Extension) 
This extension can help you automatically translate incoming/outgoing WA chats (including regional languages like Javanese/Sundanese & foreign languages) and convert foreign currency directly on WhatsApp Web instantly. 

Installation Steps: 

Download the code in ZIP form from GitHub: https://github.com/yokitea/wa_ext 
(Click the green Code button -> select Download ZIP). 

Extract the ZIP file on your computer (keep the folder in a safe place, don't delete it). 

Open the Google Chrome browser, then type chrome://extensions/ in the address bar and press Enter. 

Activate Developer mode (slide the switch in the top right corner of the page). 

Click the Load unpacked button in the top left corner of the page. 

Select the extracted folder (select the main folder). 

Open WhatsApp Web in Chrome, then Refresh (F5) your chat page. 

How to Use: 

Money Conversion: 

Price text like $50 or 100 USD automatically The Rupiah conversion appears in the chat. 

Translate In: Move the cursor (hover) to a foreign message, click the 🌐 button on the outer left side of the chat balloon. 

Translate Out: Type the chat in Indonesian, then click the 🌐 button in the type column or press Ctrl + Alt + T before sending! 

Change Language & Exchange Rate: 

Click the puzzle icon (Extension) at the top right of Chrome -> select WhatsApp Web Helper. 

Note: Only works in the Chrome browser
